# Project Ingester Development Roadmap

This document tracks the step-by-step development plan for the repository.

## 1. Project Creation

- [ ] Implement core logic for creating new projects.
- [ ] Setup project structure (folders, files).
- [ ] Integrate with Kitsu/Gazu for project initialization.

## 2. Project Querying

- [ ] Implement functionality to properly search and retrieve project details.
- [ ] Fetch project metadata, entities, and status.

## 3. Project Updation/Modification

- [ ] Allow modification of existing project details.
- [ ] Handle updates to pipeline configurations and project settings.
- [ ] Sync changes back to Kitsu/Gazu.

## 4. Project Deletion/Archival

- [ ] Implement safe deletion or archival of projects.
- [ ] Ensure data integrity and backup before removal.
- [ ] Handle associated files and entities during deletion.
